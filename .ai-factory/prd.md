# ParentShield

## Platform
앱인토스 (Vite + React + TDS)

## One-liner
자녀의 앱 사용 시간과 콘텐츠를 실시간으로 관리하고 싶지만 기존 솔루션은 복잡하거나 우회가 쉬운 문제를 해결하는 스마트 자녀보호 앱

## Problem
한국 부모들은 자녀의 스마트폰 중독과 유해 콘텐츠 노출을 걱정하지만, 기존 자녀보호 앱들은 설정이 복잡하고 아이들이 쉽게 우회할 수 있으며, 실시간 알림이 부족해 실효성이 떨어집니다. Reddit과 국내 맘카페에서 지속적으로 제기되는 불만이며, 특히 초등~중학생 자녀를 둔 부모의 80% 이상이 자녀보호 기능에 불만족합니다.

## Target Users
초등학교 고학년~중학생 자녀를 둔 한국 부모 (30~45세), 맞벌이 가정, 자녀의 디지털 활용은 허용하되 안전하게 관리하고 싶은 부모

## Core Features
1. AI 기반 유해 콘텐츠 자동 탐지 및 차단 (SNS 대화, 검색어, 앱 콘텐츠 분석)
2. 앱별/카테고리별 세밀한 시간 제한 설정 (게임 1시간, 교육 무제한 등)
3. 자녀가 우회 시도 시 부모에게 즉시 푸시 알림
4. 자녀와 부모가 함께 보는 주간 리포트 (사용 패턴, 관심사 분석)
5. 보상 시스템 연동 (목표 달성 시 추가 시간 제공)

## Monetization
- Model: freemium subscription
- Estimated MRR: $12,000
- Reasoning: 무료 버전은 기본 시간 제한만 제공하고, 프리미엄(월 9,900원)에서 AI 콘텐츠 분석, 실시간 알림, 상세 리포트 제공. 한국 초중학생 자녀 보유 가구 약 300만 중 1%만 전환해도 3만명, 그 중 40%가 유료 전환 시 월 1.2억원(약 $12,000). 자녀 안전은 한국 부모의 최우선 지출 항목이며, 학원비 월 50만원 쓰는 가구가 월 1만원 자녀보호 앱에 지불 의향 매우 높음.
