# ParentShield

앱인토스 (Vite + React + TDS) 자녀의 앱 사용 시간과 콘텐츠를 실시간으로 관리하고 싶지만 기존 솔루션은 복잡하거나 우회가 쉬운 문제를 해결하는 스마트 자녀보호 앱 한국 부모들은 자녀의 스마트폰 중독과 유해 콘텐츠 노출을 걱정하지만, 기존 자녀보호 앱들은 설정이 복잡하고 아이들이 쉽게 우회할 수 있으며, 실시간 알림이 부족해 실효성이 떨어집니다. Reddit과 국내 맘카페에서 지속적으로 제기되는 불만이며, 특히 초등~중학생 자녀를 둔 부모의 80% 이상이 자녀보호 기능에 불만족합니다.
